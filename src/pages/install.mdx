	# <i class="lni lni-npm" style="position : relative; top : 8px; margin-right : 5px"></i> Installation

    For install **vitest-cucumber** it's easy : 

    ~~~bash
    npm install @amiceli/vitest-cucumber --save-dev
    ~~~~

    It's done, you don't need a config file or update *vitest* config.

    ## Run vitest

    **vitest-cucumber** uses *vitest* functions like `describe`, `test` etc 

    Now you can run vitest like you want, for example with coverage : 

    ~~~bash
    vitest --run --coverage
    ~~~

    ### Dependencies

    Since `3.4.4` vitest-cucumber use `vitest` version `>=2.0.0`.
